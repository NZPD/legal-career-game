# Ultimate Game Hub - Roblox

An all-in-one game combining three major gameplay systems.

## 🎮 Game Systems

### ⚡ Element Awakening
- Players spawn with random element (Fire, Void, Lightning, Ice, Earth, Wind)
- Level elements 1-100
- Unlock abilities at milestones
- Combine elements for combo effects

### 👑 Kingdom Conquest  
- Create or join guilds
- Capture castles and territory
- Fight guild wars
- Manage resources (Gold, Ore, Wood, Food)

### 🏹 Hunter Association
- Progress from F Rank to S Rank
- Complete missions for rewards
- Earn money and equipment
- Build your hunter legacy

## 📊 Player Progression
- Unified level-up system across all modes
- Cross-game currency (Gold)
- Equipment system
- Stat progression

## 🚀 Installation

1. Download repository as ZIP
2. Extract files  
3. Open Roblox Studio
4. Create new baseplate
5. Follow SETUP.md
