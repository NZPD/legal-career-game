# Element Awakening

## Elements
- 🔥 **Fire** - High damage, area attacks
- 🌑 **Void** - Dark magic, life steal
- ⚡ **Lightning** - Fast attacks, chain damage
- ❄️ **Ice** - Freeze effects, crowd control
- 🌍 **Earth** - Defense, shields
- 💨 **Wind** - Speed, evasion

## Progression
1. Randomly awakened with an element
2. Level it up (1-100)
3. Unlock abilities at level milestones
4. Gain skill points
5. Combine elements for combos
